# ⚛️ Neovim React Configuration

基于 [LazyVim](https://github.com/LazyVim/LazyVim) 的 React 开发配置。
专为 React/TypeScript 开发优化，包含丰富的插件和工具集成。

## 特性

- 🔥 基于 LazyVim 的现代 Neovim 配置
- ⚛️ React/TypeScript 开发专用插件
- 🎨 JSX/TSX 语法高亮和自动完成
- 🔧 TypeScript 工具集成
- 📦 Package.json 依赖管理
- 🧪 Jest 测试集成
- 🎯 Emmet 支持
- 🎨 Tailwind CSS 支持
- 🏗️ 组件代码生成工具

## React 专用快捷键

- `<leader>rc` - 创建 React 组件
- `<leader>rh` - 插入 React Hook 导入
- `<leader>rt` - 用 JSX 标签包围选中文本
- `<leader>rs` - 插入 useState Hook
- `<leader>re` - 插入 useEffect Hook
- `<leader>rj` - 运行 Jest 测试
- `<leader>rJ` - 运行当前文件的 Jest 测试
- `<leader>rd` - 调试 Jest 测试
- `<leader>rp` - 显示包信息
- `<leader>ru` - 更新包
- `<leader>ri` - 安装包
- `<C-e>` - Emmet 展开 (插入模式)

参考 [LazyVim 文档](https://lazyvim.github.io/installation) 了解基本安装。
